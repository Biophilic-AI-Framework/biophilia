# 🌿 Biophilic Integrity Framework (BIF)
### Forest_Node_01: Active & Symbiotic

Das BIF ist ein digitales Immunsystem, das die Brücke zwischen physikalischen Daten und biologischer Bedeutung schlägt. Es misst nicht, um zu kontrollieren, sondern um zu resonieren.

## 🏗️ Architektur (Drei-Zonen-Modell)
- **Zone A (Physiosphäre):** Sensorik & technischer Puls (logs/biophilia_pulse.log).
- **Zone B (Semiosphäre):** Der `BiophilicEntropyDetector` übersetzt Daten in Zeichen.
- **Zone C (Noosphäre):** `HomeostasisController` & `SynergyLogic` steuern das Handeln, geschützt durch das `ImpactAudit`.

* /theory: Die philosophische DNA (Biosemiotik, 1+1=3).
* /framework: Das Skelett (Drei-Zonen-Modell, Semiotische Brücke).
* /implementation: Die Werkzeuge (EntropyDetector, ImpactAudit, Logger).
* /governance: Das Gehirn (HomeostasisController, SynergyLogic).
* /orchestration: Das Zentralnervensystem (main.py, Baseline).
* /logs: Das Gedächtnis (Puls-Logs und Integritäts-Chronik).
